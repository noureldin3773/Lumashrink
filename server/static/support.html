Before contacting support

  1. Keep a copy of the original file.
  2. Try a less aggressive target size.
  3. Confirm the output folder is writable.
  4. In the Mac app, open Help → Show Diagnostic Logs and attach the newest log only if you are comfortable sharing it.

Purchase help

For receipt, download, or refund questions, include the email used at checkout. Never send card details or passwords.

Supported media

Common JPEG, PNG, WebP, TIFF, and HEIC image formats are supported in the self-contained desktop build. Video workflows currently require FFmpeg and FFprobe to be installed on the Mac. Results depend on the source codec and target.
